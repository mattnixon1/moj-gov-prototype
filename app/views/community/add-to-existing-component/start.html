Add to the Spinner component

All new components submitted to the design system must include the following details:

  • Component name
  • Overview description of the component
  • How the component is being used in your service
  • What problem the component solved
  • An image file to show what the component looks like

This information is required to provide users with a basic understanding of the component and how to use it.

Providing your personal information

Your personal information is also required. This includes your:

  • name
  • email address
  • job role
  • team name

The team will use this information for only 2 reasons:

  • To contact you about the component if required for any reason.
  • To include on the component page, with your permission, when publishing the new component live.

Adding further information about the component

If you have more than the minimum required information about the component available, you can add this after you've submitted it.

You will see an option to "Add more information". Select this and you'll be taken to the "Add to an existing component" page where you can add more information.

The MoJ Design System team will then review this as with all community submissions.
