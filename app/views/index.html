More info about this prototype

V1 was built using a copy of the MOJ Frontend.

For V2 we were adding dynamic pages that was not a feature for the MOJ Frontend so we added MOJ Frontend elements onto the GOV.UK design system.

There were quite a number of problems and conflicts getting the side navigation to work which means this isn't as neat as most prototypes.

Some of the styles were getting overwritten so there are some additional styles both embedded and inline (sorry).

There were also issues including the MOJ javascript libraries and making them happy with the GOV.UK scripts.

The side navigation is a working but slightly clunky version compared to the MOJ one, it does work but not as well as the proper one.

By the time V4 is being worked on there should be a way to pass data around forms on the MOJ Frontend kit so there should be no need to continue on this prototype.

Should you need to add another version on here there will be a readme file in the main app folder.
