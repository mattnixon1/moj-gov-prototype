Add information to an existing component

After a 'community' component has been published live on the design system, you, and other users, have the opportunity to continue developing it.

This is done by users continuing to add information and resources to the component.

Additional information you can add to the component

Further information you can provide about the component includes:

  • accessibility findings
  • research findings
  • code for the component such as HTML, Javascript, CSS, SASS and Nunjucks
  • links to Figma files
  • additional image or supporting files
  • any additional information not listed

You also have to provide your:

  • name
  • email address

How do you add information to a component

To add further information about a component, you need to select the:

  • component to update from the community component list in the side navigation menu
  • "Add more component information" link in the component banner

You'll then have the opportunity to add information about the component including:

  • accessibility findings
  • research findings
  • code for the component such as HTML, Javascript, CSS, SASS and Nunjucks
  • links to Figma files
  • additional image or supporting files
  • any additional information not listed

You then submit the information for review.

Design team review of component information

The MoJ Design System team will then review the new information about the component. This can take up to 5 working days.

If there are no issues and the team does not need to contact you, the information will be added to the component in the design system.
